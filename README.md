# Capstone Expense Tracker

This project is a simple expense tracking application built with Python. It provides a Streamlit web interface for managing and visualizing expenses.

## Features
- Add, view, and manage expenses
- CSV-based data storage
- Streamlit web app for interactive visualization
-

## Files
- `app.py`: Streamlit web application
- `expenses.csv`: CSV file storing expense records

## Getting Started

### Requirements
- Python 3.8+
- Streamlit

### Installation
1. Clone the repository or copy the project files.
2. Install dependencies:
   ```bash
   pip install streamlit
   ```

### Usage
#### Web App
Run the Streamlit app:
```bash
streamlit run app.py
```



## Data Format
The `expenses.csv` file should have columns like:
- Date
- Category
- Amount
